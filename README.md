# Data Cleaning & Reporting Automation

## Project Overview

This project automates the process of data cleaning, quality assessment, KPI generation, reporting, and dashboard creation using Python and Power BI. The goal is to improve reporting efficiency by reducing manual preprocessing and generating automated business insights from retail sales data.

---

## Objective

* Perform automated data quality checks
* Detect missing values and duplicate records
* Clean and preprocess data
* Generate KPI reports automatically
* Create visual summaries and business insights
* Build an interactive Power BI dashboard

---

## Dataset

**Global Superstore Dataset**

The dataset contains retail sales transactions from multiple countries and markets.

### Dataset Information

* Records: 51,290+
* Columns: 27
* Features include:

  * Order Date
  * Ship Date
  * Sales
  * Profit
  * Quantity
  * Discount
  * Customer Information
  * Product Information
  * Region and Market Details
